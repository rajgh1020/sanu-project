📖 About Game Library Tracker

🎯 Project Purpose

This web app was built to help gamers track their games — what they’ve finished, what they’re currently playing, and what they want to play. It's meant to be simple, useful, and fun to use.

🛠 Technologies Used

  • HTML – Page structure
  • CSS – Styling and layout
  • JavaScript – Interactivity (form logic, filter, validation)
  • Git & GitHub – Version control and hosting

🧩 Features Overview

  • Add, view, and remove games with status
  • Filter games in the Gallery
  • Use the Tracker page to manage your current list
  • Send feedback via the Contact page

👨‍💻 About the Developer

I'm an IT student and gamer who built this project to combine what I love with what I’ve learned. It was created entirely from scratch as a final assignment using no frameworks.
